# Weather App

A sleek and responsive Weather Application built using HTML, CSS, and JavaScript. This app allows users to get real-time weather updates for any city worldwide.

## Features
- Real-time Data: Get live temperature and weather conditions instantly.
- Responsive Design: Optimized for both desktop and mobile viewing.
- Detailed Info: Displays humidity, wind speed, and weather descriptions.
- Search Functionality: Quickly search for weather data by city name.

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- API: OpenWeatherMap API (or the weather service you used)

## How to Use
1. Clone or download this repository.
2. Open `weather.html` in your favorite web browser.
3. Enter a city name and hit search!
